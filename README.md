
# NFCU Dashboard

A Simple dashboard that show's a user's Navy Federal banking information.

## Tech Stack

**Client/Frontend:** React + TypeScript + Vite + React Bootstrap

**Server/Backend:** Express.js for now, but eventually ASP.NET

**HTTP Client:** Axios (frontend and backend)

**Banking Integration:** Plaid API

**Validation:** Zod

**Build:** Vite + SWC

**Package Manager:** npm

**Version Control:** Git

**Editor:** VS Code

**Environment Management:** dotenv
